Help
Usage
Press the "scan" button and allow camera usage if requested. Next, you can scan any qr code, containing a spotify link by aranging it inside the highlighted area of the camera live feed. Doing so will open the song in the background. You can then use the back arrow button to load the previous song, the play button to start/stop the current song or the scan button to scan a different code.
Connect Spotify
Since this website is using spotify as the song provider, you need to be connected to spoitfy in order to play the full songs. If you do not have a premium account, or are not connected to it, you will only be able to listen to excerpts of the songs.
